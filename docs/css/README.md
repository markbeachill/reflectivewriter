# AI chat CSS files

These files style example chat transcripts. They come from a separate piece of
software (an "AI chat" HTML converter) and are kept deliberately apart from the
main site stylesheet (`../style.css`).

Every rule is scoped under `.ai-chat-page`, so a chat block can be dropped into a
page that already has its own CSS without the two interfering. This is scoping,
not full isolation; for guaranteed isolation use an iframe or Shadow DOM.

## Chat-style CSS

- `aichat.css` — default chat style (used by the example pages).
- `aichat-default.css` — identical to `aichat.css`, included for explicit naming.
- `aichat-minimal.css` — minimal chat style.
- `aichat-dark.css` — dark chat style.
- `aichat-warm.css` — warm chat style.

## Word-friendly CSS

- `aichat-word.css` / `aichat-word-default.css` — default Word-friendly style.
- `aichat-word-plain.css` — plain Word-friendly style.
- `aichat-word-compact.css` — compact Word-friendly style.

## How to use

Link the main site stylesheet first, then one chat stylesheet:

```html
<link href="../style.css" rel="stylesheet"/>
<link href="../css/aichat.css" rel="stylesheet"/>
```

Then wrap a transcript in the chat markup:

```html
<section class="ai-chat-page" aria-label="Example chat">
  <div class="ai-chat-window">
    <div class="ai-chat-log">
      <section class="ai-chat-turn ai-chat-turn--user">
        <div class="ai-chat-role">User</div>
        <div class="ai-chat-bubble"> ... </div>
      </section>
      <section class="ai-chat-turn ai-chat-turn--chatbot">
        <div class="ai-chat-role">Chatbot</div>
        <div class="ai-chat-bubble"> ... </div>
      </section>
    </div>
  </div>
</section>
```

The example pages under `../examples/` are generated by
`scripts/build_examples.py`, which writes exactly this markup. Because the chat
CSS is external, it is not regenerated by the site build; update the files here
directly if you want to change the chat look.
